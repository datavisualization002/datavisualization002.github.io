For a figure like  [Continuous-Year-Geography](docs/continuous-year-geography/), user needs to specify a grid. There are many predesigned grid could be found in [Flourish examples](https://flourish.studio/examples/?Industry=Featured), including states within US, countries of Europe. More comprehensive grid could be found in this [json file](https://raw.githubusercontent.com/hafen/grid-designer/master/grid_list.json) provided with [geofacet](https://cran.r-project.org/web/packages/geofacet/vignettes/geofacet.html) package in R. 
Details on `geofacet` could be found [here](https://cran.r-project.org/web/packages/geofacet/vignettes/geofacet.html).
